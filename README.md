# Netflix Content Analysis

An exploratory data analysis project examining Netflix's catalog of movies and television shows. The analysis explores trends in content type, ratings, release years, genres, and other characteristics through data cleaning, visualization, and descriptive statistics.

## Project Overview

Streaming platforms produce large collections of content across many genres, ratings, and release periods. This project analyzes the Netflix catalog to identify trends in available content while demonstrating a complete exploratory data analysis workflow using Python.

## Features

- Exploratory analysis of Netflix content
- Data cleaning and preprocessing
- Descriptive statistics
- Distribution and comparative visualizations
- Analysis of content characteristics and trends
- Interpretation of relationships among key variables

## Technologies

- Python
- pandas
- NumPy
- Matplotlib
- Jupyter Notebook
